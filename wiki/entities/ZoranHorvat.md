# Zoran Horvat

Principal Consultant at Coding Helmet, speaker, and independent .NET trainer. Author of 100+ articles and numerous Pluralsight video courses on object-oriented design, functional programming in C#, design patterns, and clean coding.

## Known Work
- Pluralsight courses including *Collections and Generics in C#*, *Making Your C# Code More Object-oriented*, *Making Your C# Code More Functional*, *Design Patterns in C# Made Simple*, and *Writing Purely Functional Code in C#*.
- Articles on Coding Helmet, including [[what-makes-list-so-efficient-in-dotnet]] and [[using-record-types-in-cs]].
- Courses on design patterns, refactoring, and iterative OOD.

## Recurring Themes
- Idiomatic, modern [[CSharp]] usage — leveraging language features like [[CSharpRecords]] to reduce boilerplate.
- Explaining the runtime cost of common .NET abstractions (e.g., [[ListOfT]], [[AmortizedAnalysis]]).
- Object-oriented and functional design as complementary, not opposed.
