# C#

Microsoft's primary statically-typed, object-oriented language for the [[DotNet]] platform. Recent versions have added features that pull the language toward immutable, functional, and value-oriented styles — notably:

- **C# 9** — [[CSharpRecords]] (reference-type), `init`-only property setters, the `with` expression for [[NonDestructiveMutation]].
- **C# 10** — `record struct`, allowing records to be value types.
- **.NET 6** — the `DateOnly` / `TimeOnly` types frequently used as record components.
