# .NET

Microsoft's managed runtime and class library, host to languages like [[CSharp]], F#, and VB.NET. Its Base Class Library (BCL) ships standard collections including [[ListOfT]], arrays, dictionaries, and queues. Recent platform versions (.NET 5+) co-evolve with [[CSharp]] features such as [[CSharpRecords]] and types like `DateOnly`/`TimeOnly` (added in .NET 6).

## Connections
- [[ListOfT]] — flagship resizable collection in the BCL.
- [[CSharp]] — primary language targeting the runtime.
- [[CSharpRecords]] — language/runtime feature requiring .NET 5+.
- [[ZoranHorvat]] — prominent author/trainer in the .NET ecosystem.
